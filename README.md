# Lubrum.github.io
A static web site developed mainly with Jekyll, HTML5, CSS, SASS and JS.

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run start
```

### Compiles and minifies for production
```
npm run build
```
